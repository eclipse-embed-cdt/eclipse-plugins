STM32F0xx Standard Peripherals Library update history

  • The STM32F0xx Standard Peripherals Library is a complete package, consisting of device drivers plus 67 examples for 22 different peripherals and template project for 5 different IDEs.

V1.2.0 / 22-November-2013

Main Changes

  • Add the support of STM32F030 value line devices

V1.1.0 / 31-July-2013

Main Changes

  • Add support of STM32F0xx Low-density devices (STM32F0XX_LD devices, STM32F050xx and STM32F060xx microcontrollers where the Flash memory ranges between 16 and 32 Kbytes)
  • Add template project for Raisonance (RIDE) toolchain
  • Miscellaneous update in Peripheral and EVAL board drivers and examples
