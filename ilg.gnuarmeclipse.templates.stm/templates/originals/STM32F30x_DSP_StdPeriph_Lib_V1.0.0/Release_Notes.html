STM32F30x/31x DSP and Standard Peripherals Library update History

  • The STM32F30x/31x DSP and Standard Peripherals Library is a complete package, consisting of device drivers plus a set of examples for all of the standard device peripherals, for STM32F30x and STM32F31x devices 32-bit Flash microcontrollers.
  • The STM32F30x I2C CPAL  is a library which provides a high layer API for I2C peripherals communication. For more details please refer to "STM32F0xx and STM32F3x I2C communication peripheral application library (UM1566)" available for download on www.st.com/stm32f3
  • It contains also the CMSIS DSP Software Library, which feature a suite of common signal processing functions for use on Cortex-M processor based devices. The library is completely written in C and is fully CMSIS compliant. High performance is achieved through maximum use of Cortex-M4F intrinsics. For more details, refer to CMSIS DSP Software Library
For complete information about the Library, please refer to the User Manual "Description of STM32F30xx/31xx Standard Peripheral Library (UM1581)" available for download on www.st.com/stm32f3.
